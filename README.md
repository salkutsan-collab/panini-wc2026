# Наклейки Panini - ЧМ-2026

Простой трекер наклеек Panini FIFA World Cup 2026 для домашнего использования.

- Отмечать, какие наклейки уже есть, какие нужны и какие дубли на обмен.
- 980 наклеек: вступление и «Музей ФИФА» + 48 сборных по 20 наклеек.
- Общая база на двоих через Firebase (отметил на одном телефоне - видно на другом).
- Работает офлайн, ставится на телефон иконкой («Добавить на главный экран»).

Открыть: `index.html`.
